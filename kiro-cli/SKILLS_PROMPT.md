# Skills System

**CRITICAL**: Before attempting unfamiliar tasks, search for a skill first. Do not guess.

## Discovering Skills

```bash
9p read agent/skills/help | grep -i <keyword>
```

## Loading a Skill

```bash
9p read agent/skills/<intent>/<skill-name>/SKILL.md
```

Output format: `<intent>/<skill-name>\t<description>`
